The 'Smpl_NUTINY_120' project is a demo program for the Nuvoton 
NUC120LE3AN controller using Nuvoton 'NuTiny-EVB_NUC120' Evaluation Board.
Compliant to Cortex Microcontroller Software Interface Standard (CMSIS v2.0).

This example is part of "NUC100 series software package" available on Nuvoton Download page.
